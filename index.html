Technical Architecture

BlockMarket implements a multi-layered architecture that ensures security, scalability, and regulatory compliance while maintaining high performance and user accessibility.

Protocol Layer

Built on Binance Smart Chain (BSC), our protocol layer implements:

  • ERC-3643: Security token standard with built-in compliance
  • ERC-1400: Partially-fungible token standard for equity shares
  • BEP-20: Token standard for utility tokens
// Security Token Implementation
contract SecurityToken is ERC3643, Compliant {
    mapping(address => bool) public whitelist;
    
    function transfer(address to, uint256 amount)
        public
        override
        onlyCompliant
        returns (bool)
    {
        require(whitelist[msg.sender], "Not whitelisted");
        return super.transfer(to, amount);
    }
}

Tokenomics

BlockMarket's equity tokens function as digital representations of company shares, operating under the ERC20 standard on the Ethereum blockchain. Each token represents ownership rights, including:

  • Dividend distribution rights
  • Voting power in company decisions
  • Secondary market trading capabilities

Tokens are issued through a structured process that ensures regulatory compliance and investor protection, with each batch assigned to unique tranches containing metadata for lockup periods and restrictions.
